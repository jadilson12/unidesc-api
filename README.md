## Descrição

Uma api que foi desenvolvida para utilizar no aplicativo Ionic

## Requisito funcional

- crud aluno
- crud professor
- crud turma

## Requisito não funcional

- Gerenciamento api com swagger

## Requerimento nescessário

- Node 10.x
- NPM 6.x

## Instalação do cli do nestjs

```bash
$ npm i -g @nestjs/cli
```

## Baixar o projeto

```bash
$ git clone https://github.com/jadilson12/unidesc-api
$ cd unidesc-api
$ npm install
```

## Configurando o banco de Dados mysql

Caso seja necessário alterar acessa o arquivo /src/app.modules

Configuração default:

- host: 'localhost',
- port: 3306,
- username: 'root',
- password: '123456',
- database: 'unidesc',

## Iniciar o projeto

```bash
$ npm install
$ npm run start

```

## endpoints

```bash
$ http://localhost:3000/
```
